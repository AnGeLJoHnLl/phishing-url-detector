🔐 Phishing URL Detector

Aplicación web desarrollada con Python que detecta URLs de phishing utilizando Machine Learning.

---

🚀 Tecnologías
- 🐍 Python  
- 🤖 Scikit-learn  
- 🌐 Streamlit  
- 📊 Pandas  

---

🧠 Descripción
Este sistema analiza características de una URL (longitud, patrones sospechosos, uso de IP, etc.) para determinar si puede ser phishing o segura utilizando un modelo de Machine Learning (Random Forest).

---

⚙️ Funcionalidades
- 🔍 Análisis de URLs en tiempo real  
- ⚠️ Predicción de phishing  
- 📈 Probabilidad de riesgo  
- 🎨 Interfaz web moderna e interactiva
